# Fxkeb

Get USD from KRW based on fx rate infomation from fxkeb.

## Installation

Add this line to your application's Gemfile:

```ruby
# gem 'fxkeb'
gem 'fxkeb', git: 'git://github.com/jinto/fxkeb.git'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fxkeb

## Usage

Fxkeb.to_usd(1000) # => 0.9
